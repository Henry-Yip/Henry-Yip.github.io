    Appendix A

    Proposition: An Ellipse with semi-major axis a and semi-minor axis b has area \(\pi ab\)

    Proof: Consider Green’s Theorem (Stoke’s Theorem on \(xy\) plane):

    \[\oint_C(P d x+Q d y)=\iint_D\left(\frac{\partial Q}{\partial x}-\frac{\partial P}{\partial y}\right) d x d y\]

    We aim to find polynomials such that \(\frac{\partial Q}{\partial x}-\frac{\partial P}{\partial y}=1\)

    Say \(P(x,y)=-\frac{1}{2}y\) and \(Q(x,y)=\frac{1}{2}x\)

    So we can estimate the area (denote by \(D\)) as:

    \[D=\frac{1}{2} \oint_C(x dy-ydx)\]

    Standard parameterization of an ellipse:

    \[{\displaystyle (x,y)=(a\cos(t),b\sin(t))\quad {\text{for}}\quad 0\leq t\leq 2\pi }\]

    \(\begin{align} D &=\frac{1}{2} \oint_C(x dy-ydx)\)
    &= \int_0^{2\pi} (acost)(bcost) dt - (bsint)(-asint)dt \end{align}$$

    Using \(sin^2t+cos^2t=1\), we can quickly see D= \(\pi a b\). Q.E.D.
